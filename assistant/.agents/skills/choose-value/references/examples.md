# Базовый выбор

Замени `<ЗаголовокФормыВыбора>` на заголовок, возвращенный `ПолучитьИзмененияОкна` или `ПолучитьЭлементыАктивногоОкна`.

Шаг 1: открой форму выбора и осмотри её.

```bsl
// ^^^ Некоторый код выше ^^^

Выбрать ( "#Контрагент" ); // Открывает форму выбора для поля
возврат ПолучитьЭлементыАктивногоОкна (); // Осматривает новое открытое окно перед действием
```

Шаг 2: выполни поиск внутри осмотренной формы выбора.

```bsl
Подключить ();
Здесь ( "<ЗаголовокФормыВыбора>" );
ВнестиЗначение ( "#СписокСтрокаПоиска", "Рога и копыта" ); // Использует стандартный поиск по списку
возврат ПолучитьИзмененияОкна (); // Осматривает суженный набор результатов перед чтением строк
```

Шаг 3: подтверди выбор найденной строки.

```bsl
Подключить ();
Здесь ( "<ЗаголовокФормыВыбора>" );
КСтроке ( "#Список", "#Код", "000001402" ); // Активирует нужную строку
Нажать ( "#ФормаВыбор" ); // Подтверждает выбор и закрывает форму выбора
возврат ПолучитьЭлементыАктивногоОкна (); // Осматривает исходную форму, которая снова стала активной
```
