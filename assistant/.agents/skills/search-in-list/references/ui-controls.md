## Очистка расширенного поиска (удаление примененного поиска)

Чтобы удалить примененный ранее расширенный поиск:

```bsl
Нажать ( "#ФормаОтменитьПоиск" );
```

Элемент `ФормаОтменитьПоиск` может различаться в зависимости от метаданных формы.

## Очистка назначенных/быстрых фильтров

В некоторых списках могут быть специальные фильтры, расположенные в верхней части списка. Обычно ID этих фильтров начинается с `<Поле>Filter`, где `Поле` - имя фильтруемого поля. Например, `СкладFilter`, `ВалютаFilter` и так далее. Также, в некоторых случаях суффикс `Filter` может отсутствовать или выглядеть как некий уникальный идентификатор, назначенный системой. Перед очисткой таких фильтров прочитай их значения и реши, имеет ли очистка смысл.

## Расширенная фильтрация

Наряду с расширенным поиском многие списки предоставляют фильтрацию через диалог `Настроить список...`.

### Открытие настроек фильтра

Обычно есть кнопка с ID `ФормаНастройкаСписка` (часто с заголовком `Настроить список...`). ID может различаться; при необходимости вызови `ПолучитьЭлементыАктивногоОкна` и проанализируй форму.

### Детали редактора фильтров

Диалог обычно включает такие вкладки: `Основное`, `Отбор`, `Сортировка`, `Условное оформление`, `Группировка`. Для фильтрации данных используй таблицу `КомпоновщикНастроекПользовательскиеНастройкиЭлемент*Отбор`

Типичные действия:

- Добавить строку фильтра:
  - `КомпоновщикНастроекПользовательскиеНастройкиЭлемент*ОтборДобавитьЭлементОтбора`
- Удалить строку фильтра:
  - `КомпоновщикНастроекПользовательскиеНастройкиЭлемент*ОтборУдалить`

В каждой строке обычно есть такие колонки:

1. `КомпоновщикНастроекПользовательскиеНастройкиЭлемент*ОтборЛевоеЗначение` - поле, по которому фильтровать
2. `КомпоновщикНастроекПользовательскиеНастройкиЭлемент*ОтборВидСравнения` - вид сравнения
   (обычно: `Равно`, `Не равно`, `Заполнено`, `Не заполнено`, `Содержит`, `В списке`, `Не в списке`, `Больше`, `Больше или равно`, `Меньше`, `Меньше или равно`)
3. `КомпоновщикНастроекПользовательскиеНастройкиЭлемент*ОтборПравоеЗначение` - значение отбора

### Применение фильтров

Чтобы применить настроенный отбор, нажми `ФормаЗакончитьРедактирование` (Завершить редактирование)

## Сброс фильтров к значениям по умолчанию

Некоторые списки запоминают ранее примененные фильтры и настройки. Если ожидаемые данные не удается найти, попробуй восстановить значения по умолчанию. Для этого, выполни следующее:

1. Открой настройки списка `ФормаНастройкаСписка`.
2. Нажми `ФормаСтандартныеНастройки` (Установить стандартные настройки)

## Чтение содержимого таблицы

После применения поиска можно прочитать результат с помощью:

```bsl
возврат ПолучитьСодержимоеТаблицы("#Список");
```

Используй возвращенные данные таблицы, чтобы определить результат поиска и при необходимости, если записей несколько, найти лучших кандидатов
